# ATM Machine Simulation using Python

## 📌 Project Overview

This project is a simple **ATM Machine Simulation** developed using Python.
It demonstrates basic banking operations such as:

* PIN creation
* Cash deposit
* Cash withdrawal
* Balance checking
* PIN changing

The project is completely console-based and designed for beginners who want to practice Python fundamentals like:

* Functions
* Loops
* Conditional statements
* User input handling
* Match-case statements
* Variable management

---

## 🚀 Features

### 🔐 Secure PIN Setup

* User must create a valid 4-digit PIN
* PIN confirmation is required

### 💰 Deposit Money

* Add money to the account balance

### 💸 Withdraw Money

* Withdraw cash from available balance
* Prevents overdraft transactions

### 📊 Balance Inquiry

* Displays current account balance

### 🔄 Change PIN

* Allows user to change ATM PIN securely

### 🏦 Savings Account Access

* Simulates ATM banking flow with account verification
